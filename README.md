# OpenSourcePlayGround

### 背景
本项目旨在学习iOS各开源库的使用，期间也会阅读一些优秀开源库的源码，提升自己解决问题的能力，同时也是久违的从零搭起一个APP的有趣实践。
### API
数据由[TMDB](https://developers.themoviedb.org/3/getting-started/introduction)提供
后面可能会看情况接入更多的API
### 目前用到的开源库:
- AFNetworking
- CYLTabBarController
- LookinServer
- MBProgressHUD
- MJRefresh
- Masonry
- SDWebImage
- YPNavigationBarTransition
- YTKNetwork
- YYKit
